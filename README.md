# Astro Starter Kit:

Plantillas de Astro optimizada y extendida con un ecosistema completo para desarrollo profesional y flujos de automatización locales: TypeScript (Modo Estricto), ESLint, Prettier (con auto-ordenado de clases de Tailwind CSS e importaciones), EditorConfig, daisyUI v5, Astro Icon y gestión de tipografías optimizadas con Fontsource.

## Plantilla Base

Esta plantilla se basa en el Starter Kit oficial de Astro con algunos complementos

- Prettier
  - TailwindCSS
  - Astro
  - Sort Imports
  - Sort Json
- Eslint
  - Astro
  - TailwindCSS
  - TypeScript
- Iconify
- DaisyUI
- FontSource con Roboto

## Como usar esta plantilla

```bash
#En caso de que hayas hecho un fork de este repo
# usando npm
npm create astro@latest -- --template [usuarioGit]/[nombreRepositorioGit]/base

#usando pnpm
pnpm create astro@latest -- --template [usuarioGit]/[nombreRepositorioGit]/base

#Ejemplo
npm create astro@latest -- --template wcervini/astrotemplate/base
```
